Study Guide

1. Introdução ao phpMyAdmin e MySQL

O que é um Banco de Dados?

Um banco de dados é um arquivo que permite acessar informações de maneira fácil e organizada. MySQL é um dos melhores bancos de dados gratuitos.

Sistema de Banco de Dados

Composto por dados armazenados em um banco de dados, o software de gerenciamento de banco de dados (SGBD), e os programas de aplicação. As tabelas armazenam as informações dentro do banco de dados.

Utilizando o phpMyAdmin

phpMyAdmin: Ferramenta para gerenciar MySQL.

Criando um Banco de Dados:

  1. Acesse o phpMyAdmin.
  2. Digite o nome do banco de dados e clique em "Criar".
  3. Crie tabelas com campos específicos (nome, tipo, tamanho, etc.).

Estrutura das Tabelas

Campos: Nome, Tipo (Int, Varchar, Text, Date, etc.), Tamanho, Extra (auto_increment), Primária. 


CREATE TABLE tb_mural (
    id_mensagem INT(5) AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(50),
    mensagem TEXT,
    data DATE,
    hora TIME
);

Criando Códigos

HTML/PHP para inserir e exibir informações do banco de dados.

2. Funções em PHP

Definindo Funções


function nome_da_funcao($arg1, $arg2) {
    // Comandos
    return $valor;
}

Passagem de Parâmetros

Por Valor:


function mais5($numero) {
    $numero += 5;
}

Por Referência:


function mais5(&$numero) {
    $numero += 5;
}

Valores Padrão


function teste($texto = "testando") {
    echo $texto;
}

Escopo e Contexto

Variáveis Globais e Locais:


$texto = "Testando";

function teste() {
    global $texto;
    echo $texto;
}

teste(); // imprime "Testando"

3. Estruturas de Controle e Repetição em PHP

Estruturas de Decisão

if, else, elseif:


if ($num % 2 == 0) {
    echo "Par";
} else {
    echo "Ímpar";
}

Decisão Ternária:


echo ($num % 2 == 0) ? "Par" : "Ímpar";

Estruturas de Repetição

for:


for ($i = 0; $i < 10; $i++) {
    echo $i;
}

while:


while ($i < 10) {
    echo $i;
    $i++;
}

do-while:


do {
    echo $i;
    $i++;
} while ($i < 10);

4. Conexão com Banco de Dados MySQL

Conectando ao Banco de Dados


$conexao = mysqli_connect("localhost", "root", "") or die("Conexão falhou");
mysqli_select_db($conexao, "Visitas") or die("Seleção do banco de dados falhou");

Executando Consultas

Consultando Dados:


$query = "SELECT * FROM livro ORDER BY nome";
$resultado = mysqli_query($conexao, $query);
while ($linha = mysqli_fetch_array($resultado)) {
    echo $linha['nome'];
}

Inserindo Dados


$query = "INSERT INTO livro (nome, localizacao, mensagem, data) VALUES ('$nome', '$localizacao', '$mensagem', NOW())";
mysqli_query($conexao, $query);

5. PDO (PHP Data Objects)

Introdução ao PDO


$pdo = new PDO("mysql:host=localhost;dbname=aula", "root", "");

Executando Consultas

Sem Parâmetros:


$stmt = $pdo->query("SELECT * FROM agenda");
while ($row = $stmt->fetch()) {
    echo $row['nome'];
}

Com Parâmetros:


$stmt = $pdo->prepare("SELECT * FROM agenda WHERE nome = :nome");
$stmt->bindParam(':nome', $nome);
$stmt->execute();

Inserindo, Atualizando e Deletando Dados

Inserindo:


$stmt = $pdo->prepare("INSERT INTO agenda (nome, telefone, email) VALUES (:nome, :telefone, :email)");
$stmt->bindParam(':nome', $nome);
$stmt->bindParam(':telefone', $telefone);

$stmt->execute();

Atualizando:


$stmt = $pdo->prepare("UPDATE agenda SET telefone = :telefone WHERE nome = :nome");
$stmt->bindParam(':nome', $nome);
$stmt->bindParam(':telefone', $telefone);

Deletando:


$stmt = $pdo->prepare("DELETE FROM agenda WHERE nome = :nome");

6. Projeto Completo: Sistema de Cadastro de Veículos

Criando o Banco de Dados e Tabelas


CREATE DATABASE VeiculosDB;
USE VeiculosDB;
CREATE TABLE veiculos (
    codVeiculo INT AUTO_INCREMENT PRIMARY KEY,
    nomeVeiculo VARCHAR(100),
    marcaVeiculo VARCHAR(100),
    placaVeiculo VARCHAR(8),
    corVeiculo VARCHAR(15)
);

Página de Cadastro


if ($_SERVER['REQUEST_METHOD'] == 'POST') {
    $nomeVeiculo = $_POST['nomeVeiculo'];
    $marcaVeiculo = $_POST['marcaVeiculo'];
    $placaVeiculo = $_POST['placaVeiculo'];
    $corVeiculo = $_POST['corVeiculo'];

    $conn = mysqli_connect("localhost", "root", "", "VeiculosDB");
    $sql = "INSERT INTO veiculos (nomeVeiculo, marcaVeiculo, placaVeiculo, corVeiculo) VALUES ('$nomeVeiculo', '$marcaVeiculo', '$placaVeiculo', '$corVeiculo')";
    mysqli_query($conn, $sql);
    mysqli_close($conn);
}

Página de Consulta


if ($_SERVER['REQUEST_METHOD'] == 'POST') {
    $placaVeiculo = $_POST['placaVeiculo'];

    $conn = mysqli_connect("localhost", "root", "", "VeiculosDB");
    $sql = "SELECT * FROM veiculos WHERE placaVeiculo = '$placaVeiculo'";
    $result = mysqli_query($conn, $sql);
    if (mysqli_num_rows($result) > 0) {
        $veiculo = mysqli_fetch_assoc($result);
    } else {
        $veiculo = false;
    }
    mysqli_close($conn);
}

Página de Alteração e Exclusão


if ($_SERVER['REQUEST_METHOD'] == 'POST') {
    $codVeiculo = $_POST['codVeiculo'];
    $nomeVeiculo = $_POST['nomeVeiculo'];
    $marcaVeiculo = $_POST['marcaVeiculo'];
    $placaVeiculo = $_POST['placaVeiculo'];
    $corVeiculo = $_POST['corVeiculo'];

    $conn = mysqli_connect("localhost", "root", "", "VeiculosDB");
    $sql = "UPDATE veiculos SET nomeVeiculo='$nomeVeiculo', marcaVeiculo='$marcaVeiculo', placaVeiculo='$placaVeiculo', corVeiculo='$corVeiculo' WHERE codVeiculo=$codVeiculo";
    mysqli_query($conn, $sql);
    mysqli_close($conn);
}
